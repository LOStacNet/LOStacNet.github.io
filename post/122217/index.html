MC-生电服务器客户端使用

本文为上一篇文章中搭建的服务器各种功能使用教程，以及相应的客户端mod使用教程。

环境信息

  • MC版本：1.20.1
  • Fabirc：0.16.9

整合包下载地址：百度

  • LO的服务器生电客户端——简单生电
  • LO的服务器风景游玩整合——光影+遥远地平线视觉震撼
  • LO的服务器Optifine整合——同上的mod但是光影和遥远地平线不能同时开(Optifine时代眼泪)

服务器插件与配置

原版配置

比较重要的配置信息如下：

  • view-distance视距为10
  • white-list白名单开启
  • difficulty难度为困难

MCDR本体

使用!!help来显示所有命令的帮助信息。

部分命令需要一定权限执行，可咨询腐竹。

Prime Backup备份插件

自动备份和手动备份，可以在执行高危风险进行手动备份。

在控制台输入!!pb查看命令，最主要的命令有make和back，分别为手动备份和回档。

Seed种子

使用!!seed获得服务器种子。

Loaction Maker坐标共享

服务器共享路径点。

使用!!loc命令。

Here我在这里

向其他玩家显示自己的位置。

使用!!here命令。

allow_suicide允许kill

使用!!kill。

Showlt展示物品和获得头颅

获得头：!!skull <player>

展示物品：!!show

服务器MOD功能

Easy Auth

为服务器添加了离线账号管理功能，正版用户无需关心。

  • /login <password>, /l <password> - 登陆账号
  • /logout - 退出登陆
  • /register <password> <password> - 注册账号
  • /account - 管理账号
  • /unregister <password> - 注销账号
  • /changePassword <old password> <new password> - 更改密码

忘记密码或者登录出现问题请联系腐竹。

Skin Restorer

让离线玩家也可以使用皮肤。

使用方法：

  • 正版皮肤：/skin set mojang <username> -为正版玩家名
  • ely.by皮肤：/skin set ely.by <username> -使用Ely.by网站的皮肤
  • 网页皮肤：/skin set web (classic|slim) "<url>"-使用的皮肤文件，classic|slim选择粗还是细皮肤
  • 刷新皮肤：/skin refresh
  • 清除皮肤：/skin clear
  • 重置皮肤(变回默认)：/skin reset

Easy WhiteList白名单

基于用户名的白名单，联系腐竹添加游戏名。

Servux

发送结构边框信息，让客户端可以使用MiniHUB的完整功能。

Syncmatica共享原理图

实现服务器原理图共享。

使用方法

客户端在已经加载的原理图列表按下Shift并点击Share即可上传到服务器。

其他玩家打开View Syncmatica点击Download即可加载，用Unload删除。

Litematica Printer投影打印机

模组地址

前置：MaLiLib,Litematica

一个能拯救玩家生命的MOD，能非常方便的放置投影方块。

使用方法

默认情况通过CAPS_LOCK切换功能。

如果要切换配置如速度和范围，可以在Litematica设置中的通用选项页面底部找到，同时在热键选项下可以重设快捷键。

跳过放置的物品

砂轮，头颅，告示牌，发光地衣，藤蔓

物品展示框，盔甲架

如果其他方块放置错误，可以调低速度。

Tweakeroo

模组地址

前置：MaLiLib

灵魂出窍，快速点击，伪潜行，潜影盒查看，关闭粒子等等优化和辅助。

使用方法

使用X+C打开配置。

常用功能：

Fastrightclick-快速右键

Fastleftclick-快速左键

Free camera-灵魂出窍

FakeSneaking-伪潜行

Lavavisibility-水下呼吸对岩浆有效

ShulkerBoxDisplay-shift查看潜影盒

tweakMapPreview-shift查看地图

tweakPlayerInventoryPeek-查看指向玩家的背包

tweakRenderInvisibleEntities-将隐身的实体渲染为半透明

tweakRepairMode-获得经验时轮流修复背包内物品

tweakSwapAlmostBroken-工具要损坏时自动替换

tweakHandRestock-手上物品用完或者损坏时，自动补充

tweakAccurateBlockPlacement-允许在有carpetmod的情况下以固定方向放置方块

tweakPrintDeathCoordinates-死亡时自动发送坐标

disableBlockBreakingParticles-关闭方块破坏粒子

disableDeadMobRendering-关闭怪物死亡动画

disableEntityRendering-关闭实体渲染

disableFallingBlockEntityRendering-关闭下落方块实体渲染

disableParticles-关闭粒子

disablePortalGuiClosing-取消传送门关闭GUI

Masa Gadget

模组地址

前置：MaLiLib Fabric API MagicLib

假人背包，村民附魔书渲染，村民冷却渲染，僵尸村民转换预览，修复精确放置，Masa配置大全，masa配置收藏，多人游戏容器预览，

使用方法

使用G+C打开设置界面。

常用的功能：

  • 渲染村民第一本附魔书，包括交易冷却
  • 渲染僵尸村民转换时间
  • 修复精确放置协议
  • 快速切换MasaMod配置界面
  • 收藏常用的MasaMod配置
  • 多人游戏容器预览(实现假人背包查看)
